# **Scripts**

**clone the repo**

```bash
git clone https://github.com/Sam44323/Pusher.git
```

---

## **Run the backend**

---

**Move to the server dir**

```bash
cd server
```

**Install the dependencies**

```bash
go get or go get ./...
```

---

**populate the env's**

```bash
cp .example.env .env

```

---

**Running the dev-server**

```bash
go generate
```

---

## **Run the frontend**

**Move to the frontend-dir**

```bash
cd client
```

**Install the dependencies**

```bash
yarn
```

---

**populate the env's**

```bash
cp .example.env .env

```

---

**Running the dev-server**

```bash
yarn dev
```
